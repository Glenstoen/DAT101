1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. [You should use constants whereever the valus is not supposed to change. Variables should be used when the value is expected to change. Not all values can be constants, for example a counter in a loop needs to be a variable.]

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. [parentheses affects the order of operations, the parentheses are calculated first, followed by multiplication and division etc.]

  3. How do we limit floating-point numbers to a given number of decimals?

    4. [You can use the toFixed() method to limit floating-point numbers to a given number of decimals.]

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. [You can compare two floating-point values by rounding them to a certain number of decimal places using the toFixed() method, and then comparing the rounded values.]

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. [Comparison operators return either TRUE or FALSE. You can check the result of a comparison operator by using it in an if statement or by logging the result to the console or printing it out.]

  6. How does the Math.random() function work, and what can it be used for?

    7. [Math.random will give a random number. It can be used for generating random numbers, or selecting random items from a list.]

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. [Modulus (%) is an operator that returns the remainder of a division operation. It is useful for determining if a number is even or odd.]
